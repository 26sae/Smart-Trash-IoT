## Stack
- Flutter 3.x (Android / iOS)
- Firebase Auth — email/password, no self-registration
- Cloud Firestore — bin metadata, sensor logs, reports, users
- Firebase Realtime Database — live Arduino fill level
- Provider — state management
- Google Fonts — Syne, DM Sans, DM Mono

## Setup (do these in order)

### 1. Prerequisites
```bash
flutter --version        # needs 3.x
node --version           # needs 18+
npm install -g firebase-tools
dart pub global activate flutterfire_cli
```

### 2. Create Flutter project and copy files
```bash
flutter create smart_trash
cd smart_trash
# Replace lib/ and pubspec.yaml with the files from this zip
```

### 3. Create Firebase project
- Go to https://console.firebase.google.com
- Add project → name it SmartTrash → disable Analytics → Create
- Enable Authentication → Email/Password
- Create Firestore Database → test mode → pick region
- Create Realtime Database → test mode → pick region

### 4. Connect Firebase to Flutter
```bash
flutterfire configure
# Select your SmartTrash project
# Select android (and ios if needed)
# This overwrites lib/firebase_options.dart with your real keys
```

### 5. Install packages
```bash
flutter pub get
```

### 6. Seed Firebase data

**Authentication — add 2 users manually:**
- admin@tipqc.edu.ph / admin123
- janitor@tipqc.edu.ph / janitor123

**Firestore — create these documents:**

Collection: `users`
- Document ID = UID of admin (copy from Auth tab)
  ```
  name: "Admin User"
  email: "admin@tipqc.edu.ph"
  role: "Administrator"
  ```
- Document ID = UID of janitor
  ```
  name: "Janitor Staff"
  email: "janitor@tipqc.edu.ph"
  role: "Janitor"
  ```

Collection: `bins`
- Document ID: `BIN-001`
  ```
  id: "BIN-001"
  location: "TIP QC Canteen"
  wasteType: "General"
  fillLevel: 0
  lastCollected: <timestamp: now>
  sensorOnline: true
  ```

**Realtime Database:**
```
bins → BIN-001 → fillLevel: 0
```

### 7. Run
```bash
flutter run
```

## Arduino (HC-SR04 → Firebase RTDB)

```cpp
#include <WiFi.h>
#include <Firebase_ESP_Client.h>

#define WIFI_SSID     "YOUR_WIFI"
#define WIFI_PASSWORD "YOUR_PASSWORD"
#define DATABASE_URL  "https://your-project-default-rtdb.firebaseio.com"
#define TRIG_PIN  5
#define ECHO_PIN  18
#define BIN_HEIGHT_CM 30.0   // measure your actual bin interior height

// In loop():
float duration = pulseIn(ECHO_PIN, HIGH);
float distance = duration * 0.034 / 2;           // cm from sensor to trash
float fillPct  = (1 - distance / BIN_HEIGHT_CM) * 100;
fillPct = constrain(fillPct, 0, 100);
Firebase.RTDB.setFloat(&fbdo, "/bins/BIN-001/fillLevel", fillPct);
delay(300000);  // every 5 minutes
```

Install library: **Firebase Arduino Client Library for ESP8266 and ESP32** by Mobizt

## Troubleshooting
| Problem | Fix |
|---|---|
| `google-services.json` missing | Re-run `flutterfire configure` |
| Firebase permission denied | Set Firestore + RTDB to test mode in Console |
| App crashes on launch | Verify `await Firebase.initializeApp()` is in `main()` |
| `flutter pub get` fails | Run `flutter pub cache repair` then retry |
| Emulator not found | VS Code → Ctrl+Shift+P → Flutter: Launch Emulator |
