# Smart-Trash-IoT

Description: This application helps households track trash segregation and sends
reminders for garbage collection using IoT-enabled smart trash bins.

Technologies Used: Flutter, Dart, IoT, Firebase

Features: 
Automated segregation of non-biodegradable and biodegradable waste
Trash bin fullness detection
Garbage collection reminders
Recycling activity tracking
Weekly and monthly reports


Installation Instructions
Prerequisites
Cloning the repository
Installing dependencies
Firebase configuration
Running the app locally
Building for production

Setup
IoT Device Setup: Hardware requirements, wiring diagram, programming ESP32, sensor calibration
Cloud Services Setup: Complete Firebase setup with Authentication, Firestore, Cloud Messaging, Storage, and security rules
